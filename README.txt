Glacial Lakes of Kyrgyzstan (2016 & 2024)

Overview

This dataset contains information on glacial lakes in Kyrgyzstan for the years 2016 and 2024. It includes both tabular data summarizing lake characteristics and shapefiles representing lake locations. The data support studies of glacier-lake interactions and glacial lake evolution over time.

Files

Shapefiles

Glacial_lakes_KYRG_2016.shp : Vector file of glacial lakes in 2016 ((UTM Zone 43N / WGS84, EPSG:32643)
Glacial_lakes_KYRG_2024.shp : Vector file of glacial lakes in 2022–2024 ((UTM Zone 43N / WGS84, EPSG:32643)

Excel Tables

KYRG_Glacial_lake_2016.xlsx : Summary table of glacial lakes in 2016-2017
KYRG_Glacial_lake_2024.xlsx : Summary table of glacial lakes in 2022–2024
KYRG_Glacial_lake_evolution_16_24.xlsx : Comparative table showing glacial lake changes between 2016 and 2024
